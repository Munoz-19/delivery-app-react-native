# 📦 Delivery App – Full Stack Mobile System

Mobile delivery registration system built with React Native (Expo) and Node.js.

## 🚀 Features

- Delivery registration
- Photo capture
- Digital signature
- Checklist validation
- REST API
- PostgreSQL database

## 🛠 Tech Stack

Frontend:
- React Native
- Expo

Backend:
- Node.js
- Express
- PostgreSQL

## ⚙️ Installation

### Mobile
npm install
npx expo start

### Backend
npm install
npm start
